CS Mod Bot¶

This is a discord moderation bot made specifically for Coding Server in response to attacks by the British Empire. It currently has the following features:

  • Automatic virus scanning

  • Message purging

  • Warns and kicks

Source code is privately available at its GitHub repository. For support and questions, contact xxyl#9999 on discord.
